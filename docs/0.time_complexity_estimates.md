## Time Complexity Estimates

Some useful estimates assuming a time limit of one second:

| Input Size | Required Time Complexity |
| ---------- | ------------------------ |
| n ≤ 10     | O(n!)                    |
| n ≤ 20     | O(2^n)                   |
| n ≤ 500    | O(n³)                    |
| n ≤ 5000   | O(n²)                    |
| n ≤ 10⁶    | O(n log n) or O(n)       |
| n is large | O(1) or O(log n)         |
